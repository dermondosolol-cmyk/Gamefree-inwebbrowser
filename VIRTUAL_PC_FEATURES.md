## How to Use

### Searching for Games:
1. Click the Start Menu
2. Select "Search Games"
3. Type a game name (e.g., "Flappy", "Snake", "2048")
4. Click on any result to launch

### Loading Custom Games:
1. Click the Start Menu
2. Select "Open Browser"
3. Paste a game URL in the URL bar
4. Click "Load" or press Enter

1. Click the Start Menu
